# Сайт Социал-демократической партии «С.О.В.» ("СОВесть")

Данный проект представляет собой веб-сайт политической партии, разработанный на платформе **ASP.NET Core 9.0 (MVC)**.

## 🚀 Описание проекта
Сайт создан по техническому заданию и включает в себя:
- Адаптивную верстку (Mobile-First) с использованием Bootstrap 5 и кастомного CSS (Flexbox, Grid).
- Современный и утонченный дизайн в цветовой гамме "с белым, синим, красным и золотым".
- Страницы: Главная, О партии, Программа, Команда, Законопроекты, Контакты.
- Интеграцию векторного логотипа и тематических иконок (FontAwesome).

## 💻 Технологии
- **Фреймворк:** ASP.NET Core MVC (C# / .NET 9.0)
- **Фронтенд:** HTML5, CSS3, Bootstrap 5.3, Razor Pages (.cshtml)
- **Иконки:** FontAwesome 6

## 🛠️ Как запустить локально

1. Убедитесь, что у вас установлен **.NET SDK** (версии 8.0 или 9.0). Вы можете скачать его с [официального сайта Microsoft](https://dotnet.microsoft.com/download).
2. Откройте терминал (PowerShell / Command Prompt) в корневой папке проекта (`SovParty`).
3. Выполните команду:
   ```bash
   dotnet watch run
   ```
4. Откройте браузер по адресу `http://localhost:5000` (или по предложенному консолью адресу `https://localhost:XXXX`).

## 📁 Структура проекта
- `Controllers/HomeController.cs` - содержит логику маршрутизации по страницам.
- `Views/Home/*` - Razor-представления (.cshtml) для каждой страницы.
- `Views/Shared/_Layout.cshtml` - главная обёртка сайта (шапка, меню, подвал, подключение стилей).
- `wwwroot/css/site.css` - файл с дизайн-системой (переменными цветов, стилизацией кнопок, карточек и т.д.).
- `wwwroot/images/` - хранилище статических изображений и логотипа.
